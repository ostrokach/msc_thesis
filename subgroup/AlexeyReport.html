Comments to the Author

This manuscript presents the ELASPIC web server which can be used to predict and analyze the effect of mutations on protein domains and domain-domain interactions. This topic is of great importance. The web server is functional and very user-friendly (I particularly appreciated the available help, downloads and software, and all the available options including an HTML5 alternative to Java-based visualization).
